# 如何向 AI 描述你的 PLC 需求

## 黄金法则：越详细，结果越准确

AI 不是读心术。一个模糊的需求只能得到通用的示例代码。只有提供完整的控制规格，才能得到**可直接下载到 PLC 运行**的程序。

---

## 5 步描述法

### 第 1 步：明确 PLC 型号

❌ 错误："帮我写个 PLC 程序"
✅ 正确："Panasonic FP-XH，32 点输入/24 点输出"

**为什么重要**：不同型号支持的功能块、内存地址范围不同。

---

### 第 2 步：列出完整的 I/O 清单

```
输入：
- X0: 启动按钮（常开）
- X1: 停止按钮（常闭）
- X2: 急停（常闭）
- X10: 物料检测传感器

输出：
- Y0: 电机接触器
- Y1: 报警灯
```

**关键**：标注每个 I/O 的常开/常闭类型，这直接影响梯形图中的触点选择。

---

### 第 3 步：用步骤列表描述动作流程

❌ 错误："控制一个电机"
✅ 正确：
```
1. 按下启动按钮 → 电机启动
2. 松开启动按钮 → 电机保持运行（自锁）
3. 按下停止按钮 → 电机停止
4. 急停触发 → 立即停止，故障灯亮
5. 故障复位后才能重新启动
```

---

### 第 4 步：明确安全条件

```
安全要求：
- 急停必须切断所有输出
- 安全门打开时禁止动作
- 过载保护触发后需手动复位
- 两个气缸不能同时前进（互锁）
```

---

### 第 5 步：补充特殊要求

```
额外要求：
- 启动前需要蜂鸣器预警 3 秒
- 记录今日产量（计数器）
- 支持手动/自动两种模式切换
- 故障时发送报警信号到上位机
```

---

## 示例：完整的需求描述

```
PLC 型号：Panasonic FP-XH

I/O 配置：
输入：
  X0: 启动按钮（常开）
  X1: 停止按钮（常闭）
  X2: 急停（常闭）
  X3: 热继电器（常闭）
  X10: 气缸 A 前位磁性开关
  X11: 气缸 A 后位磁性开关
输出：
  Y0: 电机接触器
  Y1: 报警灯
  Y2: 气缸 A 前进电磁阀
  Y3: 气缸 A 后退电磁阀

动作流程：
1. 按下启动 → 电机启动
2. 电机运行 5 秒后 → 气缸前进
3. 气缸前位检测 → 延时 2 秒 → 气缸后退
4. 气缸回位 → 电机停止 → 等待下一次启动

安全要求：
- 急停/热继电器触发立即停止
- 气缸前进和后退电磁阀互锁
- 故障后必须按复位按钮才能重启
```

这样的描述，AI 生成的代码准确率可达 90%+。
