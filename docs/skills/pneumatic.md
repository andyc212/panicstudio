# SKILL: 气动控制

## 场景描述

气缸顺序控制是自动化设备中最常见的执行机构控制。涉及电磁阀驱动、磁性开关检测、顺序互锁等。

## 标准 I/O 配置

| 地址 | 类型 | 名称 | 说明 |
|------|------|------|------|
| X0 | INPUT | Start_PB | 启动按钮 |
| X1 | INPUT | E_Stop | 急停 |
| X10 | INPUT | CylA_FWD | A缸前位磁性开关 |
| X11 | INPUT | CylA_BWD | A缸后位磁性开关 |
| X12 | INPUT | CylB_FWD | B缸前位磁性开关 |
| X13 | INPUT | CylB_BWD | B缸后位磁性开关 |
| Y0 | OUTPUT | CylA_FWD_Y | A缸前进电磁阀 |
| Y1 | OUTPUT | CylA_BWD_Y | A缸后退电磁阀 |
| Y2 | OUTPUT | CylB_FWD_Y | B缸前进电磁阀 |
| Y3 | OUTPUT | CylB_BWD_Y | B缸后退电磁阀 |

## 典型动作流程

1. 启动 → A缸前进（Y0）→ 检测前位 X10
2. A缸到位 → B缸前进（Y2）→ 检测前位 X12
3. B缸到位 → 延时 1秒 → B缸后退（Y3）
4. B缸后退到位 X13 → A缸后退（Y1）
5. A缸后退到位 X11 → 循环等待下一次启动

## 关键安全要求

- **互锁**：A缸前进时绝对禁止后退电磁阀同时得电
- **超时保护**：每个动作设置最大时间限制，超时报警
- **原位检测**：启动前必须确认所有气缸在后位

## AI Prompt 模板

```
请编写双气缸顺序控制程序，PLC 型号 FP-XH。

I/O：
- X0: 启动按钮
- X1: 急停
- X10-X13: A缸/B缸前后位磁性开关
- Y0-Y3: A缸/B缸前进后退电磁阀

动作顺序：A前进→B前进→延时1秒→B后退→A后退→等待

安全要求：
1. 每个气缸前进/后退电磁阀互锁
2. 每个动作设置5秒超时保护
3. 急停时所有电磁阀立即断电
4. 启动前检测所有气缸是否在后位
```
