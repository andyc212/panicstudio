# SKILL: 传送带系统

## 场景描述

多段传送带联动控制，常用于物流分拣、生产线传输。需要处理传送带启停顺序、物料检测、分拣动作等。

## 标准 I/O 配置

| 地址 | 类型 | 名称 | 说明 |
|------|------|------|------|
| X0 | INPUT | Start_PB | 系统启动 |
| X1 | INPUT | Stop_PB | 系统停止 |
| X2 | INPUT | E_Stop | 急停 |
| X10 | INPUT | Sensor_A | A段物料检测 |
| X11 | INPUT | Sensor_B | B段物料检测 |
| Y0 | OUTPUT | Belt_A | A段传送带电机 |
| Y1 | OUTPUT | Belt_B | B段传送带电机 |
| Y2 | OUTPUT | Pusher | 分拣推杆电磁阀 |

## 典型动作流程

1. 启动系统 → A段传送带先启动 → 延时 2秒 → B段启动
2. 物料到达 A段传感器（X10）→ 记录计数
3. 物料到达 B段传感器（X11）→ 判断是否需要分拣
4. 需要分拣 → 推杆动作（Y2）→ 延时收回
5. 停止系统 → B段先停 → 延时 3秒 → A段停止（防止物料堆积）

## AI Prompt 模板

```
请编写两段传送带联动控制程序，PLC 型号 FP-XH。

I/O 配置：
- X0: 启动按钮
- X1: 停止按钮
- X2: 急停
- X10: A段物料传感器
- X11: B段物料传感器
- Y0: A段传送带电机
- Y1: B段传送带电机
- Y2: 分拣推杆电磁阀

动作要求：
1. 启动时 A段先启动，延时2秒后 B段启动
2. 停止时 B段先停，延时3秒后 A段停
3. 物料在 B段检测到时，如为不良品则推杆分拣
4. 记录总产量和不良品数量
5. 任意段故障时全系统停止
```

## 常见错误

- ❌ 两段同时启停（会导致物料堆积或拉空）
- ❌ 推杆持续时间不足（物料未完全推出）
- ❌ 没有物料排队缓冲逻辑
