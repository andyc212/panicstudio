# SKILL: 安全互锁系统

## 场景描述

工业设备安全是红线。本 SKILL 涵盖安全门、光栅、双手按钮、双手启动等安全控制逻辑。

## 安全等级分类

| 等级 | 描述 | 典型应用 |
|------|------|---------|
| CAT B | 基础安全 | 单通道急停 |
| CAT 2 | 诊断覆盖低 | 安全门监控 |
| CAT 3 | 诊断覆盖中 | 双手按钮 |
| CAT 4 | 高诊断覆盖 | 光栅+安全继电器 |

## 标准 I/O 配置

| 地址 | 类型 | 名称 | 说明 |
|------|------|------|------|
| X0 | INPUT | E_Stop_1 | 急停按钮 1（NC）|
| X1 | INPUT | E_Stop_2 | 急停按钮 2（NC）|
| X2 | INPUT | Safety_Door | 安全门关闭（NC）|
| X3 | INPUT | Light_Curtain | 光栅正常（NC）|
| X4 | INPUT | TwoHand_1 | 双手按钮 1 |
| X5 | INPUT | TwoHand_2 | 双手按钮 2 |
| Y0 | OUTPUT | Safety_Relay | 安全继电器 |
| Y1 | OUTPUT | Alarm_Horn | 报警蜂鸣器 |

## 双手按钮启动逻辑（关键）

**要求**：
1. 两个按钮必须在 0.5秒内同时按下
2. 只按一个按钮不触发动作
3. 松开任一按钮立即停止
4. 每次启动必须重新双手同时按

## AI Prompt 模板

```
请编写安全互锁控制程序，PLC 型号 FP-XH。

安全输入：
- X0, X1: 双急停按钮（常闭）
- X2: 安全门关闭检测（常闭）
- X3: 光栅正常（常闭）
- X4, X5: 双手启动按钮

输出：
- Y0: 安全继电器（控制设备电源）
- Y1: 报警蜂鸣器

安全要求：
1. 任一急停触发立即切断 Y0
2. 安全门打开时切断 Y0
3. 光栅被遮挡时切断 Y0
4. 双手按钮必须在 0.5秒内同时按下才允许启动
5. 松开任一按钮立即停止
6. 所有安全故障必须手动复位后才能重启
7. 添加安全状态指示灯
```

## 常见错误

- ❌ 使用自锁逻辑锁定双手按钮（违反安全规范）
- ❌ 安全复位自动进行（必须人工确认）
- ❌ 忽略光栅响应时间（通常 < 15ms）
- ❌ 没有差异时间检测（双手不同步）
